# Records

Each file in this directory should document one technical topic from problem discovery through design choice and implementation outcome.

## Recommended structure

1. Background
2. Problem and impact
3. Current state analysis
4. Options considered
5. Tradeoff comparison
6. Final decision
7. Implementation design
8. Testing and validation
9. Outcome and follow-up

## Naming

Use `YYYY-MM-DD-topic.md`.
