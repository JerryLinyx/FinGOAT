# Archive

This directory stores frozen documentation snapshots for completed or baseline versions.

## Rules

- Each version gets its own folder, such as `v0.1.0/`.
- Archived files should not be edited casually after the version is frozen.
- Current active work belongs in `../planning/` and `../appendix/`.

## Available snapshots

- `v0.1.0`
- `v0.1.1`
- `v0.1.2` (latest baseline)
